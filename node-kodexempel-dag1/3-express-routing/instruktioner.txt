------

Installera först med npm, öppna terminlafönstret i samma mapp som du ska köra filerna.

1. Skapa package.json fil
npm init

2. Installera Express (lokalt i den mapp du har filerna)
npm install express

-----

Kör koden i ett terminalfönster i samma mapp
node app.js

Använd t ex Insomnia för att prova http-requests för (t ex post, put, delete)
http://localhost:3000

(Kolla i app.js vilka endpoints som finns och testa tillsammans med rätt metod (t ex post)
